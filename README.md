# Smart Attendance System (Java)

## Description
This project is a Smart Attendance System using Face Recognition.
It is developed using Java, JSP, and Servlets.

## Features
- Login system
- Register new face
- Train face model
- Mark attendance
- View attendance on web page

## Technology Used
- Java
- JSP & Servlet
- Apache Tomcat
- OpenCV (concept)

## Attendance Storage
Attendance is saved at:
C:/attendance/attendance.txt

## How to Run
1. Import project into Eclipse
2. Configure Tomcat server
3. Create folder C:/attendance
4. Run project on server
5. Open login.jsp
